My First Blog

##第一步
安装nodejs，安装淘宝cnpm镜像，cnpm install -g hexo-cli #安装hexo框架
##第二步
创建blog目录，进入目录，hexo init #初始化博客，hexo s 启动本地服务，4000端口查看是否成功
##第三步
hexo n “一个新博客” #新建一个博客，返回blog目录，hexo clean #清理，hexo g #生成
##第四步
github创建一个新仓库，cnpm install –save hexo-deployer-git #安装git部署插件,配置config.yml文件 #在Deploy下配置type为git repo为github仓库地址,我这里给的地址是ssh地址，branch为master
##第五步
hexo d #部署到github仓库 在这里碰到坑了，git与github连接问题，设置git config -global user.name=xxx, git config -global user.email=xxx之后，生成ssh key 复制到github的setting ssh连接里生成key里面
